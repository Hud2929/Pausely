Problem: Emails going to spam?
→ Set up custom SMTP (SendGrid/AWS SES) in Supabase
→ Add SPF/DKIM DNS records to your domain

Problem: Deep links not opening the app?
→ Make sure you're testing on a real device, not simulator
→ Check that "pausely" URL scheme is in Info.plist
→ Verify Redirect URLs in Supabase match exactly

Problem: User clicks email but not logged in?
→ Check Xcode console for error messages
→ Make sure Supabase auth is working
→ Test with auto-confirm disabled in Supabase settings
