## How It Works

### 1. Track Usage
Pausely tracks how much you use each streaming/service app:
- **Automatic**: Screen Time integration (with permission)
- **Manual**: Quick-add buttons (+30m, +1h, +2h)
- **Edit anytime**: Tap usage to adjust

### 2. Calculate True Cost
Instead of just "$15/month", Pausely shows:
- **Cost per hour**: "$5.20/hour based on 3 hours this month"
- **Value rating**: "Poor value" or "Great value!"
- **Comparison**: How you compare to typical users

### 3. Smart Suggestions
When usage is low, Pausely suggests pausing:
> 💡 "You've only used Netflix for 15 minutes this month. That's $61.96/hour! Consider pausing to save $15.49."

### 4. One-Tap Pause
Tap "Pause" and Pausely opens the service's pause page directly. No searching, no hassle.

## Using Smart Pause

### In Dashboard
1. Open Pausely → Dashboard
2. Look for **"💡 Smart Pause"** section
3. See potential savings across all subscriptions
4. Tap **"View All Suggestions"** for details

### In Subscription Detail
1. Tap any subscription
2. See **Usage Tracking** section
3. View cost-per-hour calculation
4. If usage is low, see Smart Pause banner
5. Tap to pause with one button

### Adding Usage Manually
1. Go to subscription detail
2. Tap the **pencil icon** in Usage Tracking
3. Enter hours/minutes spent this month
4. Or use quick buttons: "Haven't used", "Light usage", etc.

## The Math Behind Smart Pause

```
Cost Per Hour = Monthly Cost / Hours Used

Example:
- Netflix: $15.49/month
- You used it: 2.5 hours this month
- Cost per hour: $15.49 / 2.5 = $6.20/hour

Compare to:
- Movie ticket: ~$15 for 2 hours = $7.50/hour
- Your Netflix is actually reasonable!

But if you only used it 15 minutes:
- Cost per hour: $15.49 / 0.25 = $61.96/hour!
- That's worse than a movie ticket! Pause it!
```
